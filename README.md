# 🖥 HDMI / Display Monitor (Java + JNA)

A Java-based system tool that detects display changes such as HDMI connect/disconnect using Windows native APIs.

---

## 🚀 Features

- Detects number of connected displays
- Monitors screen resolution and position
- Listens for display change events (HDMI plug/unplug)
- Uses Windows native API via JNA
- Real-time monitoring using background thread
- Simple GUI output panel

---

## 🧠 Concepts Used

- Java Swing (GUI)
- JNA (Java Native Access)
- Windows API (User32)
- Event-driven programming
- Multithreading

---

## ⚙️ Requirements

- Java (JDK 8 or above)
- Windows OS
- JNA Libraries:
  - `jna.jar`
  - `jna-platform.jar`

---
